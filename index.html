O que são Clients?

INSTRUÇÕES: Nesta aula aprendemos sobre a parte do cliente, que é onde as resquisições são realizadas ao servidor. No clinte não é executado qualquer função, apenas apresenta as respostas enviada pelo servidor.

(Voltar)

O que são Servers?

INSTRUÇÕES: O servidor é responsavel por receber as resquisições e enviar suas respostas. Nele ficam armazenados todos os comandos e recursos necessários para o funcionamento do sistema.

(Voltar)

Como é uma estrutura básica de um HTML?

INSTRUÇÕES: O HTML é composto por TAGS que realizam a construção de um página. As tags basicas do hmtm são:

HTML

Tag que define onde começa e terminar a página html.

HEAD

Tag onde são definidas as infomações da página.

BODY

Tag que define o corpo da pagina html.
